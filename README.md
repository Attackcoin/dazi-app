# 搭子 App 🤝

> **反孤独引擎** — 用 AI 降低走出门的心理门槛，让孤独的人真正交到真朋友

## 产品定位

搭子 App 是一款线上预约、线下见面的社交应用。核心差异化：

- 🤖 **AI 全程辅助** — 语音发布、破冰话题、智能推荐、月报生成
- 🔒 **三级信任体系** — 手机号 → 活体检测 → 身份证实名
- 💰 **防鸽子押金** — 微信/支付宝担保 + 芝麻信用免押金
- 📍 **线下真实连接** — QR签到验证，爽约有记录有代价

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | FlutterFlow（iOS + Android）|
| 后端 | Firebase（Auth / Firestore / Functions / FCM / Storage）|
| 聊天 | Firebase Realtime Database |
| AI | Claude Haiku API |
| 搜索 | Algolia |
| 地图 | Google Maps API |
| 实名认证 | 阿里云实人认证 |
| 支付 | 微信支付 + 支付宝 + 芝麻信用 |

## 核心功能

- 📋 **搭子广场** — 6大分类/30细分，个性化推荐，AI匹配度评分
- 🎤 **语音AI发布** — 说一句话，10秒完成发布
- 👫 **男女配额** — 发布时设定比例，实时显示报名进度
- 💬 **富媒体聊天** — 语音/图片/视频/位置
- ⚡ **即刻出发** — 临时起意，30分钟内见面
- 🏆 **发布者勋章** — 靠谱搭子主 / 金牌搭子主
- 🎉 **搭子回忆卡** — 活动后 AI 生成，一键分享朋友圈
