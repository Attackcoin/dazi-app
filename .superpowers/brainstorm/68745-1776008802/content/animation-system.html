动画系统 — 重度反馈设计

每个操作都有视觉/触觉反馈，分为 4 个层级

层级 1：微交互（Micro — 每个点击都有感觉）

按钮点击
按下 → 缩小 0.95 + 降低亮度
松开 → 弹回 1.0 + 涟漪扩散
时长: 120ms · 曲线: easeOut
卡片点击
按下 → 缩小 0.97 + 边框加亮
松开 → 弹簧弹回
时长: 200ms · 曲线: spring(damping:15)
Tab 切换
光条滑动到新位置
图标颜色渐变 + 发光渐显
时长: 250ms · 曲线: easeInOut
触觉反馈
轻触按钮 → lightImpact
滑卡过阈值 → selectionClick
操作成功 → mediumImpact
配合 HapticFeedback API

层级 2：列表动画（List — 内容有生命感）

列表项入场
从下方 20px 淡入滑上
错位延迟：item[i] delay = i * 50ms
时长: 350ms · 曲线: easeOutCubic
骨架屏闪烁
玻璃色块 + shimmer 光带从左扫到右
光带颜色：rgba(255,255,255,0.06)
时长: 1500ms · 循环 · linear
下拉刷新
自定义刷新指示器：品牌 logo 旋转
拉动时 logo 逐渐显现 + 放大
阈值: 80px · 弹簧回弹
空状态
图标缓慢上下浮动
文字淡入 + 按钮弹入
浮动: 2s 周期 · 8px 幅度

层级 3：页面转场（Page — 空间感）

push 进入
新页从右滑入 + 轻微放大（0.95→1.0）
旧页左滑 + 缩小（1.0→0.95）+ 降暗
时长: 350ms · 曲线: easeInOutCubic
Bottom Sheet
磨砂玻璃背景淡入
Sheet 从底部弹簧弹出
背景 dim + blur 渐增
时长: 400ms · 曲线: spring
Hero 共享元素
PostCard 图片 → 详情页大图
头像 → Profile 页头像
Flutter Hero widget 自动处理
时长: 300ms · 曲线: easeInOut
Tab 页切换
横向滑动 + 淡入淡出
支持手势左右滑动切换 Tab
时长: 250ms · 曲线: easeInOut

层级 4：庆祝动画（Celebration — 高光时刻）

🎉 成功加入局
卡片飞出 + 粒子爆发效果
中心弹出 ✓ + "加入成功！"
渐变光环扩散
时长: 900ms · HapticFeedback.mediumImpact
⭐ 互评完成
星星逐个亮起 + 放大弹跳
最终评分数字弹入
彩色纸屑飘落（5星时）
时长: 1200ms
📍 签到成功
GPS 定位动画 → 锁定
✓ 图标弹出 + 脉冲波纹扩散
confetti 纸屑短爆
时长: 800ms · HapticFeedback.heavyImpact
🎴 AI 回忆卡
卡片 3D 翻转展示
背景光斑色彩流动
文字逐行打字机效果
时长: 2000ms

⚡ 性能策略

  • 伪玻璃优先：90% 的玻璃效果用 半透明底色 + 微弱边框 + 模糊阴影，不用 BackdropFilter
  • 真 blur 限制：只在 BottomSheet overlay、Modal 弹层、导航栏 3 处用真 BackdropFilter
  • 动画开关：检测 MediaQuery.disableAnimations + 低端机自动降级（去粒子、减 blur）
  • 列表动画只首屏：只有首次加载时有入场动画，滚动加载的新项直接显示
  • Hero 动画白名单：只在 PostCard→详情、头像→Profile 两个路径启用
